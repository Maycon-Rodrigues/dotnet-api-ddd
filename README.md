# dotnet-api-ddd
Learning about DDD on dotnet

## :rocket: Technologies

- **C#**

- **.Net 5**

- **MySql**

  

## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/Maycon-Rodrigues/dotnet-api-ddd.git
```

#### Backend

```bash
# Go into the repository
$ cd dotnet-api-ddd

# Run the app
$ dotnet run
```
